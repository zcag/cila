# Brief — Documentation home / landing

**Type:** docs home (entry page, not the doc body) · **Stack:** Astro

Build the documentation home for **Ledgerwave**, an open-source TypeScript library for double-entry accounting primitives. The audience is backend engineers who landed here from npm or GitHub and want, in under five seconds, to know what it is, install it, and find their path (Quickstart · Concepts · API reference · Recipes). It needs a copy-able install line, a tiny but real code sample that shows the core idea, clear cards into the four doc sections, a search affordance slot, and a version/changelog hint. Calm, legible, fast — developer-trust aesthetics over marketing flash, but still not generic. Monospace used with intent; excellent code typography and contrast.
