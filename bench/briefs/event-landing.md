# Brief — Event / conference landing

**Type:** event landing page (single page) · **Stack:** Astro

Build the landing page for **Frequency '26**, a two-day independent conference on generative audio and music technology, held in Lisbon. The page sells the ticket and the FOMO: a hero with the name, the one-line premise, dates + city, and a "Get a ticket" CTA; a speakers strip (6–8, photo + name + a one-line credential); a schedule-at-a-glance (two days, key sessions); a venue/city note that makes Lisbon part of the draw; ticket tiers (early-bird / standard / student) with the early-bird honestly framed; and a closing CTA. This is a "wow" brief — generative-audio conference, so an expressive, kinetic hero is on-theme and expected (signature motion moment, reduced-motion-safe). Energy without sacrificing legibility of the practical details (dates, price, location).
