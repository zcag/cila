# Copywriting

Write to `CONTENT.md` (positioning, core message, pillars, awareness stage, VoC swipe file). The scoring lens for everything: **Purchase = Desire − (Labor + Confusion)** — raise desire, cut effort, cut confusion.

## Frameworks (per section, chosen by awareness stage)
- **PAS** (Problem → Agitate → Solution) — problem section; colder/problem-aware traffic.
- **BAB** (Before → After → Bridge) — aspirational hero; sell the transformation.
- **FAB** (Feature → Advantage → Benefit) — feature sections; translate specs to "why you care".
- **AIDA** — whole-page arc when useful.
- **4 U's** (Useful/Urgent/Unique/Ultra-specific) — a *test* to run over every headline + CTA.
- **"So what?"** — read each line as the customer; if they'd shrug, cut or fix it.

## Page anatomy (default SaaS/lead-gen; adapt per CONTENT.md page type)
Hero → social-proof bar → Problem (PAS) → Solution / 3-step how-it-works → Features→Benefits (FAB) → testimonials w/ metrics (beside CTAs) → objections/FAQ → risk reversal (next to CTA) → final CTA → minimal footer. One primary CTA, repeated; nav stripped on focused landing pages.
- **SaaS:** lead with clarity of what it does. **App:** screenshots + store ratings + install. **Ecommerce:** buy box early, reviews above the product name, shipping/returns, guarantee near cart. **Lead-gen:** strip to the form. **Portfolio:** positioning + work as proof.

## Headlines
State a specific promise. Pass the **5-second test** (a stranger knows *what* + *why* in 5s). Harry Dry's three: **visualize it · falsify it (numbers/proof) · nobody else could say it.** Subhead does the *clarifying work* (who/how/what changes), never a restatement. Lead per the awareness stage in CONTENT.md.

## Social proof · objections · risk reversal
Specific, falsifiable testimonials (numbers > adjectives) placed **at decision moments**, beside CTAs. Defuse the top 3–5 objections (FAQ or inline). Put the guarantee / free-trial / "no card" line **adjacent to the CTA**.

## CTAs
One primary action. **Verb + value, not labor** ("Start free" not "Submit"). First-person lifts ("Start **my** trial"). 3–4 words. Friction microcopy beneath ("No credit card required"). Forms single-column, minimum fields.

## What makes it land (not generic)
Specificity, concreteness, **customer language** (use the VoC swipe file verbatim), one idea per section, scannable, clarity over cleverness. Then run the `voice` skill's anti-slop edit pass.

## Pre-ship copy checklist
- [ ] Hero passes the 5-second test; headline visualize/falsify/differentiate ✓; 4 U's ✓
- [ ] Subhead clarifies (not restates); leads per awareness stage
- [ ] One primary CTA, repeated; verb+value, first-person, ≤4 words, + friction microcopy
- [ ] Every section makes ONE point, survives "So what?"; features → benefits (FAB)
- [ ] Proof at decision moments; top 3–5 objections handled; risk reversal by the CTA
- [ ] ~6th–9th-grade reading level; VoC language; passed the `voice` anti-slop pass
